# Dhaja Allocation Tool

This is a Streamlit application for allocating bookings to dhaja allotments.

## How to Deploy on Streamlit Cloud

1.  **Push this code to GitHub:**
    *   Create a new repository on GitHub.
    *   Upload `app.py` and `requirements.txt` to that repository.

2.  **Deploy:**
    *   Go to [share.streamlit.io](https://share.streamlit.io/).
    *   Log in with your GitHub account.
    *   Click "New app".
    *   Select your repository, branch (usually `main`), and the main file path (`app.py`).
    *   Click "Deploy".

## Local Usage

To run this tool locally:

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the app:
    ```bash
    streamlit run app.py
    ```
